# Changelog

## 2.1.0 - 2026-07-18

- 历史记录和书签不再以明文 JSON 保存：新文件使用 Windows DPAPI 当前用户范围加密；首次启动会无损迁移 `history.json` / `bookmarks.json`，并尽力覆写后删除旧明文文件。
- 隐私 WebView2 目录改为跟随当前数据根目录，便携模式使用 `<程序目录>\Data\Private`；继续使用当前用户 ACL 与可用时的 EFS，同时以独立本机清理进程替代 PowerShell 守护脚本，并保留下次启动兜底清理。
- 正常退出现在会先停止导航、执行退出清除、释放全部 WebView2 控制器并等待内核子进程退出，再关闭主窗口，降低便携存储仍被 Chromium 异步 I/O 占用的风险。运行中直接拔盘或突然断电仍属于无法由应用保证写入完整性的物理中断。
- 从单文件发布物中排除未使用的 `Microsoft.Web.WebView2.WinForms.dll`；应用图标由 87,812 字节压缩到 15,534 字节，并保留 Windows 常用的 16、32、48 和 256 像素资源。
- 统一应用、用户脚本运行时、网络请求标识、全部 GitHub 多语言介绍页和官方网站的版本信息为 2.1.0，并在 GitHub 介绍页加入官方网站入口。

## 2.0.7 - 2026-07-18

- 修复启用广告拦截时，YouTube `googlevideo.com/videoplayback` 媒体请求被误拦截导致视频无法播放的问题。
- 外部播放器菜单现在可直接打开当前网页（适用于 YouTube 等分离音视频的站点），同时保留已检测媒体 URL 作为备用入口。

## 2.0.6 - 2026-07-17

- 修复动态子框架销毁时，广告拦截元素选择器在 WebView2 `Destroyed` 回调内重入解绑事件，导致 `EmbeddedBrowserWebView.dll` 触发 `0x80000003` 并终止整个浏览器的问题。

## 2.0.5 - 2026-07-17

- 新增浏览器任务管理器（`Shift+Esc`），集中显示标签页及 WebView2 进程的内存、CPU 和进程 ID，并可结束标签页或异常子进程。
- 标签页现可在标签栏中自由拖放排序，调整后的顺序会同步写入会话快照。
- 下载记录、书签与历史记录新增即时查找；书签查找可与分组筛选组合使用。
- 修复首次运行接受使用条款时，设置目录暂时不可写会通过异步启动链导致进程闪退的问题。

## 2.0.1 - 2026-07-16

- 修复分屏子菜单在 WebView2 页面上显示黑色或白色矩形伪影的问题。
- 修复深色应用配合浅色启动页强调色时，“保存”“项目地址”等主按钮文字不可见的问题。
- 重构阅读模式内容提取，限制处理的 DOM 节点、文本量与递归深度，避免复杂页面因整棵 DOM 深拷贝而闪退。
- 书签资料库现可双击书签直接在新标签页打开。

## 2.0.0 - 2026-07-16

- 修复浅色启动页和浅色强调色下主页标签、设置选中项及站点标识文字对比度不足的问题；图片/GIF 背景会采样亮度，文字面板始终提供可读对比度。
- 会话恢复改为普通标签 URL 或标签集合变化后即时防抖、单写者原子快照；保留标签顺序和重复页面，排除隐私标签，并新增“是否记录并提示恢复上次会话”设置。关闭后立即清除且停止写入，清除历史不会影响该独立记录。
- 新增阅读模式（工具栏、菜单及 F9）、全应用黑白模式、可编辑书签名称，以及包含版本、WebView2 Runtime、隐私化本地浏览器指纹和脱敏本机信息的“关于”页。
- 首次运行在创建网页进程前显示精简使用条款；同意版本仅保存于本地，拒绝后直接退出，并提供全部 10 种界面语言译文。
- 冷启动不再等待约 16 万条广告规则编译、用户脚本和历史加载；首窗显示后后台初始化，首个真实网页仍会等待拦截规则就绪。启动页图片解码和遗留隐私目录清理同样移出首帧路径。
- 新增 Windows x86 32 位兼容构建，并继续发布 x64 与 ARM64 原生构建。微软 WebView2 不提供 ARM32 Runtime/Loader，因此 Windows 10 on Arm 的 32 位兼容方式为运行 x86 版本，未用改名文件冒充 ARM32。
- GitHub 项目主页新增日语、法语、德语、韩语和繁体中文介绍，并补齐全部 10 种应用界面的 2.0.0 新功能文案。

## 1.9.6 - 2026-07-16

- “标记为广告”现支持顶层页面、同源及跨域 iframe；网页禁用原生右键菜单时提供宿主菜单兜底，并在对应子框架中立即隐藏和持续应用元素规则。
- 启动页背景色可同步为应用强调色，按钮、选中状态和交互反馈统一跟随该颜色，应用图标保持独立配色。
- 启动时不再自动打开旧页面，改为在右上角显示 10 秒会话恢复提示；会话快照统一排除隐私标签，切换到隐私标签时隐藏提示。
- 移除 x86 WebView2Loader、32 位构建提示和架构回退，只保留原生 x64 与 ARM64 发布路径。
- 增加 iframe 广告捕获及隐私会话快照边界测试，并完成双架构构建和依赖漏洞审计。

## 1.9.5 - 2026-07-16

- 补齐繁体中文、日语、韩语、葡萄牙语、西班牙语、俄语、法语和德语的全部浏览器设置与界面文案，不再依赖英语回退。
- 新增广告拦截规则订阅管理，内置 EasyList、EasyList China、CJX's Annoyance List、EasyPrivacy 和 Adblock Warning Removal List，并显示每项规则数、更新时间与错误状态。
- 支持添加和删除自定义 HTTPS 规则订阅、直接编辑 ABP 兼容自定义规则，以及手动、每天或每周更新；立即更新使用条件请求，失败时保留上次可用缓存。
- 扩展 ABP 网络规则的资源类型、站点域名、第三方和例外选项处理，并支持元素隐藏规则；网页右键菜单可将当前元素标记为广告、立即隐藏并保存站点规则。
- 发布原生 Windows x64 与 Windows ARM64 两个单文件版本，并提供 SHA-256 校验和。

## 1.9.0 - 2026-07-16

- 第三方 Cookie 站点判定改用随程序嵌入的完整 Public Suffix List，正确区分 `github.io`、`appspot.com`、`co.za` 等公共或私有后缀租户，并采用 schemeful site 边界。
- 用户脚本开始强制执行 `@grant` 和 `@connect`：未声明的特权 API 不再暴露，跨域网络请求在 C# 后端再次校验，并使用逐脚本授权令牌隔离网络请求、事件轮询、中止操作和 `GM_*Value` 私有存储。
- `GM_xmlhttpRequest` 增加 8 路并发、64 MB 内存响应和 32 MB 上传限制，避免恶意脚本耗尽内存；`GM_download` 继续流式落盘，可用于更大文件。
- 内置广告拦截新增常用 ABP/EasyList 网络规则支持，包括 `||domain^`、通配符、起止锚点、正则规则和 `@@` 例外规则；元素隐藏规则仍不在本版本范围内。
- 增加 WebView2 渲染/浏览器进程故障处理、页面自动恢复和 Runtime 更新重启提示；每 30 秒周期保存非隐私标签会话，降低异常退出后的恢复损失。
- 启动页根据背景亮度自动切换深浅文字、面板和边框；背景图片增加遮罩，修复橙、黄、浅绿、浅灰等背景下的低对比度。
- 新增 14 项隐私/权限/广告规则边界测试和 GitHub Actions，持续验证 x64、ARM64 构建与依赖漏洞。
- 发布原生 Windows x64 与 Windows ARM64 两个单文件版本。

## 1.7.1 - 2026-07-16

- 启动页背景新增红、橙、黄、浅绿 4 个常用颜色预设，现有取色器和十六进制输入保持不变。
- 浏览器界面语言扩展为简体中文、繁体中文、英语、日语、韩语、葡萄牙语、西班牙语、俄语、法语和德语，并支持按系统语言自动选择。

## 1.7.0 - 2026-07-16

- 将 WebRTC 禁用迁移到 `AddScriptToExecuteOnDocumentCreatedAsync`，在首个页面脚本前生效，并自动覆盖主文档和子框架；保护属性不可删除或重写。
- 原生 WebView2 地理位置权限现在无条件拒绝；询问和自定义坐标只通过不可配置的 document-start 模拟层返回，删除 JS 包装也无法读取真实位置。
- “阻止第三方 Cookie”改为严格模式：所有跨站请求都剥离 Cookie，响应后清除跨站写入，并将 WebView2 Tracking Prevention 提升为 Strict；设置文案明确提示可能影响跨站登录。
- 隐私资料目录新增当前用户专属 ACL、可用时 EFS 加密和独立清理监护进程；程序崩溃或被强制结束后仍会重试删除，启动时保留兜底清理。
- 重绘工具栏刷新图标，修复圆弧越界和箭头比例问题。
- 启动页背景颜色增加实时色块、系统取色器和常用颜色预设，同时保留十六进制精确输入。
- 重构 `GM_xmlhttpRequest` 为后台流式事件桥：支持上传/下载进度、readyState 1–4、超时、中止、文本/JSON/Blob/ArrayBuffer 响应、当前 WebView2 会话 Cookie，并移除 16 MB 固定响应上限。
- `GM_download` 改为 C# 后台专用文件流下载，支持 Cookie、进度、超时、中止、冲突文件名避让和临时文件清理。
- 发布原生 Windows x64 与 Windows ARM64 两个单文件版本。

## 1.5.3 - 2026-07-14

- 增加存储路径可写性/嵌套检查、翻译请求限流与分批处理、启动页自收藏防护等可靠性改进。
