# Changelog

## 1.7.1 - 2026-07-16

- 启动页背景新增红、橙、黄、浅绿 4 个常用颜色预设，现有取色器和十六进制输入保持不变。
- 浏览器界面语言扩展为简体中文、繁体中文、英语、日语、韩语、葡萄牙语、西班牙语、俄语、法语和德语，并支持按系统语言自动选择。
- 翻译目标语言改为常用语言预设下拉框，仍保留 BCP 47 自定义语言代码入口；旧版本中保存的自定义代码会自动保留。
- 发布原生 Windows x64 与 Windows ARM64 两个单文件版本。

## 1.7.0 - 2026-07-16

- 将 WebRTC 禁用迁移到 `AddScriptToExecuteOnDocumentCreatedAsync`，在首个页面脚本前生效，并自动覆盖主文档和子框架；保护属性不可删除或重写。
- 原生 WebView2 地理位置权限现在无条件拒绝；询问和自定义坐标只通过不可配置的 document-start 模拟层返回，删除 JS 包装也无法读取真实位置。
- “阻止第三方 Cookie”改为严格模式：所有跨站请求都剥离 Cookie，响应后清除跨站写入，并将 WebView2 Tracking Prevention 提升为 Strict；设置文案明确提示可能影响跨站登录。
- 隐私资料目录新增当前用户专属 ACL、可用时 EFS 加密和独立清理监护进程；程序崩溃或被强制结束后仍会重试删除，启动时保留兜底清理。
- 重绘工具栏刷新图标，修复圆弧越界和箭头比例问题。
- 启动页背景颜色增加实时色块、系统取色器和常用颜色预设，同时保留十六进制精确输入。
- 重构 `GM_xmlhttpRequest` 为后台流式事件桥：支持上传/下载进度、readyState 1–4、超时、中止、文本/JSON/Blob/ArrayBuffer 响应、当前 WebView2 会话 Cookie，并移除 16 MB 固定响应上限。
- `GM_download` 改为 C# 后台专用文件流下载，支持 Cookie、进度、超时、中止、冲突文件名避让和临时文件清理。
- 发布原生 Windows x64 与 Windows ARM64 两个单文件版本。

## 1.5.3 - 2026-07-14

- 地址栏新增自动完成下拉框：输入时即时匹配最近历史记录，并支持鼠标选择、上下键切换、回车执行和 Esc 关闭。
- 为 Bing、Google、百度和 DuckDuckGo 增加约 180ms 防抖的实时搜索联想；新输入会取消旧请求，自定义引擎和隐私标签页不会发送联想请求。
- 在下载设置中增加 Cookie 鉴权说明：外部下载器通常无法继承当前页面的 Cookie/登录会话，需要鉴权的资源建议优先使用内置下载。

## 1.5.2 - 2026-07-14

- 重做下载窗口：新增文件大小、已传输/总大小、实时进度、MIME 类型、开始与完成时间、中断原因和完整保存位置。
- 下载完成后可通过按钮或双击记录直接打开文件；无法关联打开时会在资源管理器中定位该文件。
- 将“启用开发者工具 / F12”独立突出显示在“设置 → 高级”，关闭时按 F12 会直接打开对应设置页，并明确区分 ZZZ 设置和 DevTools 内部实验设置。
- 启动 WebView2 前迁移 DevTools 偏好，自动隐藏 Edge 已废弃的 webhint 实验关闭横幅。

## 1.5.1 - 2026-07-14

- 修复网页获得焦点后 `Ctrl+F` 无法调出查找框，以及查找框被 WebView2 页面遮挡的问题。
- 分屏在较窄窗口下自动改用上下布局，并可手动切换方向；进入分屏时重置异常缩放，提供主/副页面独立缩放、界面关闭按钮和 `Ctrl+Shift+W` 快捷关闭。
- 修复点击主页时 WebView2 销毁与导航并发导致的高概率卡住问题。
- 新增书签分组：资料库可编辑、筛选分组，启动页按组展示；书签 HTML 导入导出保留文件夹分组。

## 1.5.0 - 2026-07-14

- 新增当前网页打印、保存为 PDF 以及保存为 MHT 网页归档。
- 新增隐私位置模拟：可在每次网站请求时编辑返回坐标，也可始终返回设置中的经纬度和精度；网页不会获得设备真实位置。
- 修复自定义主页图片后无法恢复默认背景的问题，并新增逐条书签主页开关、主页书签样式和宽度设置。
- 新增 F11 沉浸式全屏、双页面分屏浏览和 `Ctrl+F` 页内查找。
- 历史记录支持双击打开、删除指定记录；清除历史记录和浏览数据改为轻量的应用内现代两步确认，不再调用旧式系统警告框。
- 增加 Windows ARM64 单文件构建，嵌入 ARM64 WebView2Loader；ARM64 发布包仅随本版本提供。

## 1.2.5 - 2026-07-14

- 修复深色模式下下载、清除浏览数据等独立窗口出现浅色背景和浅色文字、内容不可读的问题。
- 完善设置、隐私、下载和资料库页面的深色控件样式，包括复选框、滑块、表格与窗口背景。
- 删除区域截图功能、菜单入口、快捷键及相关代码。

## 1.2.1 - 2026-07-14

- 翻译按钮改为双态切换：首次点击翻译，再次点击恢复翻译前保存的网页原文，第三次点击可重新翻译。
- 自动翻译后的页面同样可以通过“译”按钮恢复原文。
- Google 代理翻译模式再次点击“译”时会返回原网站，不再对翻译代理页重复翻译。

## 1.2 - 2026-07-14

- 新增原生精简启动页：只保留搜索/地址输入框，并可显示书签；可在内置启动页和当前搜索引擎网站之间切换。
- 启动页支持自定义纯色、静态图片和动态 GIF 背景；图片按显示尺寸解码，GIF 仅在可见时播放并限制帧数，启动页不会创建 WebView2 进程。
- 修正第三方 Cookie 策略：不再跨站批量删除 Cookie，仅对已识别的第三方跟踪请求移除 Cookie，避免破坏 OAuth、统一登录与支付流程。
- 新增微软 Edge 风格页内翻译，并设为默认翻译方案；保留 Google Chrome 请求模式，可按页面语言自动翻译为指定语言。
- 清除浏览数据、清除历史记录、删除用户脚本以及覆盖导入设置/规则均增加二级确认。
- 新增本机、便携和自定义数据目录三种模式；设置、书签、历史、脚本、WebView2 Cookie 与缓存可随 EXE 移动，并在重启前安全迁移。
- 增加存储路径可写性/嵌套检查、翻译请求限流与分批处理、启动页自收藏防护等可靠性改进。

## 1.1 - 2026-07-14

- 扩展用户脚本支持：元数据解析、正确运行时机、导入、匹配/排除规则、资源、持久值及常用 `GM_*` API。
- 增加一键 Google / Microsoft 网页翻译和可配置目标语言。
- 改进媒体嗅探：支持响应 MIME/内容元数据、HLS/DASH 及常见音视频格式。
- 强化隐私标签页：独立临时 WebView2 环境、仅内存脚本值、禁用持久缓存/自动填充/密码保存、拒绝站点权限、GPC 及退出清理。
- 通过单实例、共享普通 WebView2 环境、后台标签低内存模式、有限嗅探结果和标签休眠降低内存开销。
- 增加命令行 URL 处理及 `Alt+D` 地址栏快捷键。
